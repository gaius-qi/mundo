# mundo
Build vue single page application quickly based on webpack4.

## Usage

### Init new project

Use npm package

```bash
$ npm i mundo -g
$ mundo [dir]
```

Or use git clone

```bash
$ git clone git@github.com:gaius-qi/mundo.git
$ npm run fork [dir]
```

### Develop project

```bash
$ npm i
$ npm run dev

$ npm run build
``` mundo
