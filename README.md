# mundo
Build vue single page application quickly based on webpack4.

## Usage

### Use mundo init new project

```bash
$ npm i mundo -g
$ mundo [dir]
```

### Use git clone

```bash
$ git clone git@github.com:gaius-qi/mundo.git
$ npm run fork [dir]
```

### Develop project

```bash
$ npm i
$ npm run dev

$ npm run build
```
